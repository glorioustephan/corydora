# Agent Catalog

Corydora ships a normalized builtin catalog organized by category and technical lens.

## Categories

- `bugs`
- `performance`
- `tests`
- `todo`
- `features`

## Technical lenses

- `typescript`
- `react`
- `nextjs`
- `node-cli`
- `electron`
- `security`
- `database`
- `refactoring`

Imported external agents are stored as metadata under `.corydora/agents/imported-agents.json`.
